December 2017

15.12.2017 by Kurt Garloff

Updated images provided on 2017-12-15. Major changes over previous versions:

  • Linux images

    • All upstream updates, as usual
    • Use bms-network-setup.py for creating ifcfg-* from network_data.json on Bare Metal.
    • Make local_port_range an even number
    • Boot time optimizations (especially SUSE12/42)
    • Avoid cloud-init-wait-mds stalling on DNS requests

  • CentOS-7

    • kmod-20-15.el7_4.6 is broken (does break override statements), resulting in Inifiniband drivers no longer loading. Fix by downgrading to el7_4.4 until RH fixes this.
    • Note that detaching virtio (KVM ethX) network interfaces on Inifiniband flavors results in a kernel crash.
    • Lower cloud-init retries on OpenStack Network DatSource, as it stupidly does it for non-existing (404) user_data and network_data.json, resulting in slow boot times.

  • SUSE

    • Avoid waiting for interfaces as we can not distinguish between mandatory and optional interface in SLES12/openSUSE42 (wicked) any more; only wait short time and rely on cloud-init-wait-mds
    • SLES12_SPx_SAPHANA images with SAP data provider (contributed by Huawei)
    • SLES12: Add Modules and Add-On repositories, mostly disabled by default
