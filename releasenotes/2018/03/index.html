15.03.2018 by Kurt Garloff

Updated images provided on 2018-03-15. Major changes over previous versions:

Linux - New image Standard_Fedora_27_latest

  • Debian-9

    • add packages kbd and console-data (to enable ‘sudo loadkeys [de,us]‘)

  • Redhat and Oracle images with retpoline kernel

  • Oracle7 BMS image available

  • Ubuntu 16

    • includes CVE-2017-16995

  • Windows Server 2008 R2
    • Update for Windows Server 2008 R2 x64 Edition (KB4074837)
    • 2018-03 Security Monthly Quality Rollup for Windows Server 2008 R2 for x64-based Systems (KB4088875)
    • Cumulative Security Update for Internet Explorer 11 for Windows Server 2008 R2 for x64-based Systems (KB4089187)
    • Windows Malicious Software Removal Tool x64 - March 2018 (KB890830)
  • Windows Server 2012 R2
    • Windows Malicious Software Removal Tool x64 - March 2018 (KB890830)
    • 2018-03 Security Monthly Quality Rollup for Windows Server 2012 R2 for x64-based Systems (KB4088876)
    • 2018-03 Security Only Quality Update for Windows Server 2012 R2 for x64-based Systems (KB4088879)
    • Cumulative Security Update for Internet Explorer 11 for Windows Server 2012 R2 (KB4089187)

  • Windows Server 2016

    • Windows Malicious Software Removal Tool x64 - March 2018 (KB890830)
    • 2018-03 Cumulative Update for Windows Server 2016 for x64-based Systems (KB4088787)

  • Included driver and tools:

    • XEN GPL driver: 3.0.144.590 (unchanged)(XEN Images)

    • Intel SRIOV (High Performance ) driver : Version 22.0, January 27, 2017 (unchanged) - (all Images)

    • NVIDIA pGPU driver: 390.65 (unchanged) - (pGPU Images)
    • Cloudbase-Init: 9.12 (unchanged) - (all Images)
    • Intel Chipset Drivers 10.1.17479.8054 - (BMS Images)
