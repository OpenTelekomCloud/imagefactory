April 2018

11.04.2018 by Kurt Garloff

Updated images provided on 2018-04-13. Major changes over previous versions:

Linux

  • All: Include all updates from the Linxu distributors

  • SLES 11 and SLES 12:

    • Updated bms-network-setup to handle scenarios, where network devices are ordered differently than assumed in network_config.json but can not be renamed (due to conflicting names). This fixes SLES12 on physical.h2.large.
    • SAP images: Switch clocksource to tsc on xen flavors.
    • Add serial login for SLES11 BMS.
    • Set DHCLIENT_PRIMARY_DEVICE to set default route on first bond interface (SLES11).

  • EulerOS 2 (SP2):

    • Added Support for Bare Metal.
    • Resolved curcular dependencies to make grabrandompass work reliably.
    • Avoid NetworkManager on EulerOS (different from other RHEL7 family members!)
    • Prepare for switch to SP3 (May or June …)

  • Ubuntu:

    • Prepare switch over to images from ImageFactory, see https://imagefactory.otc.t-systems.com/Blog-Review/UbuntuAnnounce.html
    • cloud-init improvements have been merged upstrem (into 18.2).
